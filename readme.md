# 🛡️ Security Diagnostic Tool — Windows 11
Fortinet NSE1 Threat Landscape 3.0 Edition

## ¿Qué detecta?
- Procesos sospechosos (RATs, Backdoors)
- Conexiones C2 conocidas
- Persistencia en el sistema
- Síntomas de infección

## Instalación
pip install psutil colorama

## Uso
python security_check.py

## ⚠️ Disclaimer
Herramienta educativa y defensiva.
Solo para sistemas propios o con autorización.
